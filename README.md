# Developer Portfolio

A modern, responsive portfolio website built with Vite, React, Tailwind CSS, and GSAP animations.

## Features

- 🎨 Modern dark theme design
- ✨ Smooth animations with GSAP and Framer Motion
- 🎭 ScrollTrigger animations
- 📱 Fully responsive on all devices
- ⚡ Lightning fast with Vite
- 🚀 Optimized for performance

## Tech Stack

- **Build Tool:** Vite
- **Framework:** React 18
- **Styling:** Tailwind CSS 3
- **Animations:** GSAP, Framer Motion, AOS
- **Language:** JavaScript (JSX)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) to view the site.

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Sections

- **Hero:** Animated headline with smooth transitions
- **About:** Introduction and skills showcase
- **Projects:** Featured work with hover effects
- **Experience:** Timeline of work history
- **Contact:** Get in touch section with social links

## Performance Optimizations

- GSAP context cleanup for memory management
- ScrollTrigger `once` property to prevent re-animations
- Optimized responsive breakpoints
- Font smoothing and antialiasing
- Vite's fast HMR and optimized builds

## Customization

Update the content in each component file:
- `src/components/Hero.jsx` - Main headline
- `src/components/About.jsx` - Bio and skills
- `src/components/Projects.jsx` - Project showcase
- `src/components/Experience.jsx` - Work history
- `src/components/Contact.jsx` - Contact info
